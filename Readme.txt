This is Diwen Chen's IMGD 4300 Assignment 3 - WebGPU Intro Code.
The assignments ask to use live video, noise, and interation mechanisms.
Check out the website at: https://dchen1204.github.io/IMGD4300Assignment3/ 

Things to Know:
1. If your system does not have a webcam, just ignore the Video Background toggle. 
2. The website works in Chrome and Edge; Firebox is not supported.
3. If the website does not work in Chrome, try the following steps:
    Open a new tab in Chrome and type: chrome://flags
    In the search bar, type: WebGPU
    Look for "Unsafe WebGPU Support"
    Change the setting to Enabled
    Relaunch Chrome
